# Code of Conduct

## Our Commitment

RepoMind is a project by the HackingTheRepo team. We want contributing to this project to be a welcoming experience for everyone, regardless of age, background, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

We are committed to maintaining a respectful, professional, and harassment-free environment in all project spaces — GitHub issues, pull requests, code review comments, and any other venue where the project is discussed.

---

## Expected Behaviour

We ask all contributors, maintainers, and participants to:

- **Be respectful.** Disagree with ideas, not people. Critique code, not the person who wrote it.
- **Be constructive.** If you identify a problem, try to suggest a solution or offer to help.
- **Be patient.** Not everyone has the same background or experience level. Answer questions with the same care you'd want when you were starting out.
- **Be direct and honest.** Say what you mean clearly. Avoid passive aggression or deliberately vague feedback.
- **Assume good intent.** Most people are trying their best. If something seems off, ask for clarification before assuming bad faith.
- **Keep it on-topic.** Discussions in issues and PRs should stay focused on the technical work.

---

## Unacceptable Behaviour

The following will not be tolerated in any project space:

- Harassment, intimidation, or discrimination of any kind
- Personal attacks, insults, or derogatory comments
- Sexualised language or imagery
- Publishing someone's private information without their explicit consent
- Deliberate derailing of technical discussions
- Sustained disruptive behaviour after being asked to stop

---

## Scope

This Code of Conduct applies in all project spaces, including:

- GitHub issues, pull requests, and discussions
- Code review comments
- Any communication related to the project where one party is representing RepoMind or HackingTheRepo

---

## Enforcement

If you experience or witness behaviour that violates this Code of Conduct, please report it to the maintainers at **conduct@hackingt​herepo.dev**.

Reports will be handled with discretion. The maintainers will:

1. Acknowledge receipt of the report within 48 hours
2. Review the situation and, where necessary, reach out to the parties involved
3. Take appropriate action, which may range from a private warning to a permanent ban from the project

Maintainers who do not follow or enforce this Code of Conduct in good faith may face consequences as determined by other members of the project leadership.
